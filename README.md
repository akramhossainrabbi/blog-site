## Installation

-   `git clone https://github.com/akramhossainrabbi/blog-site.git`
-   `cd mastererp/`
-   `composer install`
-   `cp .env.example .env`
-   Update `.env` and set your database credentials
-   `php artisan key:generate`
-   `php artisan migrate`
-   `php artisan db:seed`
-   `npm install`
-   `npm run dev`
-   `php artisan serve`
